# 🧾 Split Bill

A lightweight, mobile-friendly bill splitting app for groups — no account required, no app to install.

**Live site:** https://Wendy0119-nyu.github.io/splitbill

---

## Features

- Add members and record expenses in seconds
- Flexible split — choose who shares each expense
- Auto-calculates the minimum number of transfers to settle up
- Session history saved locally in your browser
- Supports **USD, CNY, EUR**
- Available in **English, 中文, Español**

## How to Use

1. **Members** — Add everyone joining the bill
2. **Expenses** — Record each payment: who paid, how much, who splits it
3. **Settle Up** — See exactly who pays whom and how much

## Tech Stack

- Pure HTML / CSS / JavaScript — no frameworks, no dependencies
- LocalStorage for session history
- Hosted on GitHub Pages
